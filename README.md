🛒 Smart Mart Pro
Enterprise Inventory Ledger & Strategic POS Management Terminal
Smart Mart Pro  is a professional-grade Full-Stack management solution designed for modern retail environments. This system automates the mathematical complexity of wholesale-to-retail unit conversions while providing real-time strategic sales analytics for operational efficiency.
 Key Engineering Highlights
Strategic Inventory Ledger: Intelligent asset synchronization that automatically merges new inventory into existing records, preventing duplicate entries and maintaining data integrity.
Hybrid Transaction Terminal: A high-end Point-of-Sale (POS) interface that allows operators to process transactions in "Full Cases" or "Individual Pieces" with automated unit-conversion logic.
Real-Time Strategic Analytics: A live dashboard tracking Gross Revenue Generation, Aggregate Units Distributed, and identifying the Primary Performing Asset (Most Sold Item).
Operational Intelligence: Automated visual alerts (Red-coding) that trigger when inventory levels drop below critical thresholds to prevent stock-outs.
Secure Payment Simulator: Interactive transaction terminal supporting both Cash and Digital payment methods, featuring professional processing animations and instant receipt generation.
🛠️ Technology Stack
Backend: Python (Flask Framework)
Database: SQLAlchemy (SQLite) for high-integrity data persistence and relational mapping.
Frontend: HTML5, CSS3 (Enterprise Navy Design), JavaScript (ES6+).
UI/UX Components: SweetAlert2 for interactive transaction feedback and FontAwesome for professional iconography.
🔧 Installation & Operational Setup
To deploy the Smart Mart Pro terminal on your local environment, follow these steps:
Clone the Repository:
git clone https://github.com/Successlagun/Smart-Mart.git
Provision Dependencies:
pip install flask flask-sqlalchemy
Initialize the Engine:
python app.py
Access the Terminal:
Open your browser to http://127.0.0.1:5000
